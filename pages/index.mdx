# Introduction
 
Welcome to `@trytech/kml`! this is a simple library that manipulates KML files.

## What is @trytech/kml?

A **simple**, **powerful**, **flexible** and **performatic** library that allows you to read, write, validate, manipulate and convert KML files.

## Why use @trytech/kml?

- **Simple**: It is easy to use and understand.

- **Powerful**: It has a lot of features that allow you to manipulate KML files.

- **Flexible**: It allows you to read, write, validate, manipulate and convert KML files.

- **Performatic**: It is fast and efficient. In addition to processing everything in the background without working your main thread
